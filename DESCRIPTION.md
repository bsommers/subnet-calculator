# Project Description

The Subnet Calculator is designed to be a modern tool for network engineers and students. Unlike legacy calculators with dated interfaces, this tool prioritizes user experience and visual feedback.

## Key Capabilities

### IPv4 Calculator
- Real-time updates as you type.
- Visual slider for CIDR selection.
- Detailed breakdown of network parameters including binary representation for educational purposes.
- Support for Class A, B, and C presets.

### IPv6 Calculator
- Handling of standard IPv6 notation including double-colon compression.
- Prefix length calculation.
- Address expansion to full 128-bit hex representation.

### Design Philosophy
- **Aesthetics**: sleek dark mode with neon accents.
- **Usability**: clear separation of concerns (tabs) and immediate feedback.
